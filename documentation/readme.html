Movico the movico module is a scaffolding module for Drupal that improves the building of an application for Drupal The file and directory structure is based on the MVC design pattern and makes use of the object oriented features of PHP. Controllers Code that goes into the contoller directory should have the following characteristics. Controllers are the Class that is called in the uri after the initialization of Movico SITE_PATH/[application_path]/CONTROLLER/ACTION Models Models are files that manipulate the database or information store. Views views are templates and printed output that the user sees after completion of the contorller and model processes
